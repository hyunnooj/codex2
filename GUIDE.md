# 피노키오 롤플레잉 게임 가이드

이 문서는 `index.html`로 제공되는 피노키오 모험 롤플레잉 게임의 실행 방법과 진행 방식, 구조를 설명합니다.

## 실행 방법

1. 프로젝트 루트(`/workspace/codex2`)에서 아래 명령어로 간단한 정적 서버를 실행합니다.
   ```bash
   python -m http.server 8000
   ```
2. 브라우저에서 `http://localhost:8000/index.html`을 엽니다.

> 서버 없이도 파일을 직접 열 수 있으나, 일부 브라우저 보안 정책 때문에 정상 동작을 보장하기 위해 로컬 서버 실행을 권장합니다.

## 게임 진행 방법

- 화면 상단에 **용기/정직/친절/기력** 스탯이 표시됩니다.
- 각 장면은 본문 로그 영역에 누적되며, 선택지를 클릭하면 다음 장면으로 이동합니다.
- 선택지마다 스탯이 변동되고, 마지막 장면에서 누적 스탯에 따라 서로 다른 결말이 출력됩니다.
- 하단의 **처음부터 다시** 버튼으로 언제든지 초기화할 수 있습니다.

## 결말 분기

- 전체 스탯 합이 높을수록 더 긍정적인 결말로 이어집니다.
- 정직/친절 스탯이 높으면 해당 성향에 맞는 결말이 선택됩니다.

## 프로젝트 구조

- `index.html`
  - UI 레이아웃, 스타일, 게임 로직이 모두 포함된 단일 HTML 파일입니다.
  - `scenes` 배열에 각 장면의 텍스트/선택지/스탯 변화가 정의되어 있습니다.
  - `state` 객체로 현재 스탯과 진행 상황을 관리합니다.

## 커스터마이즈 팁

- 새로운 장면을 추가하려면 `scenes` 배열에 객체를 추가하고 `next` 인덱스를 연결하세요.
- 스탯 종류를 추가하려면 `state`와 `updateStats` 함수, 그리고 UI의 스탯 표시 영역을 함께 확장하세요.
